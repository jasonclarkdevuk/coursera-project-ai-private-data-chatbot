# AI Chatbot for Private Data / Resources

This practice Coursera project involves a chatbot which can generate responses to queries using information retrieval from a private PDF document.

The Flask framework is used to for the web app and server side processing of the chatbot. It also handles the management of all HTTP requests and responses.

The LangChain framework is used to help connect a RAG enhanced LLM system to the web app. This helps the chatbout understand and response to user inputs effectively and allows the system to provide various functionalities like text retrieval and summarisation.

HTML, CSS and JavaScript files were already provided for the layout, structure and interactivity of the web page / web interface.
